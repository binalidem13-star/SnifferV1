## Features

The tool features an interactive menu with the following options:
* **[1] WIFI Info:** Displays details about the current wireless connection.
* **[2] WIFI Sniffer:** Captures and analyzes packets within the local Wi-Fi network.
* **[3] Signal Sniffer:** Monitors network environments and signal strength.
* **[4] DNS Sniffer:** Logs DNS queries to track visited domains.
* **[5] Full Web scanner & Info:** Scans web targets for open ports and information.
* **[6] show wifi ports:** Lists active local ports and connections.
